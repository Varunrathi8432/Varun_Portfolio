## Architecture

```
src/
├── app/
│   ├── components/        # Navbar, Footer, Preloader, Cursor, ScrollProgress
│   ├── core/
│   │   ├── directives/    # InView, Tilt, CountUp (custom directives)
│   │   ├── models/        # TypeScript interfaces
│   │   ├── pipes/         # TruncatePipe
│   │   └── services/      # Theme, Scroll, Data services
│   ├── pages/             # Home, Blog, ProjectDetail (lazy-loaded)
│   └── sections/          # Hero, About, Skills, Projects, Experience, etc.
└── styles.scss            # Global theme variables & base styles
```

**Key architectural decisions:**

- **Standalone Components** — No NgModules, fully embracing Angular 19's modern API
- **Angular Signals** — Fine-grained reactivity with `signal()` and `computed()`
- **Custom Directives** — Reusable behaviors (scroll detection, tilt effects, count animations)
- **Feature-Based Organization** — Components grouped by domain, not type

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Varunrathi8432/Varun_Portfolio.git

# Navigate to the project
cd Varun_Portfolio

# Install dependencies
npm install

# Start development server
ng serve

# Open in browser
# http://localhost:4200
```

### Build for Production

```bash
ng build --configuration production
```
